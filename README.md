# GenAI-101

<img src="docs/images/genai101.png" align="right" width="450"  />


This is a series of courses that aims to introduce and understand the basics and foundations behind the first, to the latest advancements in the Generative AI field (Generating texts, images, multimodality…) . Using mathematical concepts and intuitions, We will explore what lies under the hood, as well as the ideas that shaped these models, and provide code handouts and exercises to help ground the concepts.
